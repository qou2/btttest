Privacy Policy

Last Updated: May 2, 2025

Introduction

This Privacy Policy explains how the Bedrock Tier Testing Bot ("we", "our", or "Bot") collects, uses, and stores information when you use our Discord bot service. We take your privacy seriously and are committed to protecting your personal information.

Information We Collect

1. User Data

  • Discord User ID: We collect and store Discord user IDs to identify users for moderation purposes.
  • Discord Username and Tag: We store usernames and tags in relation to moderation actions.
  • Server Member Information: Basic information about server members interacting with tickets or subject to moderation actions.

2. Moderation Data

Moderation Actions: We record moderation activities (warnings, mutes, kicks, bans) including:

  • Type of action taken
  • Moderator who performed the action
  • Reason for the action
  • Timestamp of the action
  • Duration (for temporary actions like mutes)

3. Ticket Information

Ticket Content: Information provided within support tickets, including:

  • Minecraft Username
  • Platform information
  • Region
  • Preferred testing gamemode
  • Any additional information provided in tickets

How We Use Information

1. Bot Functionality

We use collected information to:

  • Properly assign and manage support tickets
  • Enforce server rules through moderation features
  • Maintain records of moderation actions for accountability

2. Moderation Records

Moderation data is used to:

  • Track user behavior patterns
  • Enable moderators to make informed decisions
  • Maintain a record of administrative actions for transparency

Data Storage and Security

1. Data Storage

  • Moderation logs are stored in a JSON file on the host server
  • Data is regularly saved and backed up to prevent loss

2. Data Retention

  • Moderation records are kept for the duration of the bot's implementation in your server
  • No automatic data deletion is implemented; server administrators can manually delete the data file if needed

3. Data Security

  • We implement reasonable security measures to protect stored data
  • Access to the host server is restricted to authorized administrators only

Data Sharing

1. Third Parties

  • We do not sell or share your data with third parties
  • Your information remains within the Discord server where the bot is installed

2. Disclosure Requirements

  • We may disclose information if required to comply with a legal obligation
  • Information may be shared if necessary to protect the rights, property, or safety of users, the public, or our service

User Rights

1. Access Rights

  • Users can view their own moderation history using the bot's /modlogs command
  • Server administrators can access all stored data directly on the host server

2. Deletion Requests

  • To request deletion of your personal data, please contact the server administrators
  • Note that certain information may need to be retained for legitimate moderation purposes

Children's Privacy

Our service is not directed to children under 13. We do not knowingly collect personal information from children under 13. If you are a parent or guardian and believe your child has provided us with personal information, please contact us.

Changes to This Privacy Policy

We may update this Privacy Policy from time to time. We will notify users of any changes by posting the new Privacy Policy in the server's information channels. Your continued use of the bot after changes indicates your acceptance of the revised Privacy Policy.

Contact Information

For questions about this Privacy Policy, please contact the server administrators or moderators.
